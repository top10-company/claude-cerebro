# claude-cerebro

A inteligência de um Claude Code que trabalha bem: **método, rigor, técnica e postura.**

Este repositório **não sabe quem você é** — e é exatamente por isso que ele pode ser instalado por
qualquer pessoa, em qualquer empresa, sem carregar junto o contexto de ninguém.

```
nucleo/CLAUDE.md       como trabalhar: postura, verdade, custo, versionamento, memória
nucleo/CARREGADOR.md   as ~15 linhas que ligam isto ao Claude da sua máquina
onboarding/            o que perguntar a quem chega sem contexto — em blocos, com gatilho
dominios/              conhecimento por ofício (opcional e modular)
```

**Instalação e a regra que separa o que vai aqui do que é seu:** [`INSTALL.md`](INSTALL.md).

## A ideia

Um assistente útil precisa de duas coisas que não se misturam:

- **como trabalhar** — verificar antes de afirmar, medir antes de concluir, delegar sem colidir,
  não apagar o que é caro, discordar quando é o caso. Isso é igual para todo mundo.
- **para quem** — o que a pessoa produz, o que é sagrado, onde fica a verdade dos arquivos, quem
  aprova o quê. Isso é só dela.

Misturados no mesmo arquivo — que é como quase sempre acontece — você não consegue compartilhar o
primeiro sem entregar o segundo. Separados, a inteligência vira um bem comum que melhora com o uso
de todos, e o contexto continua privado.

Quem instala sem contexto não fica travado: o Claude percebe que não conhece o usuário e conduz o
onboarding.
